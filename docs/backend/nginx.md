### 反向代理


### nginx 常用命令
1. 查看nginx 版本号

```bash
nginx -v 
```
2. 启动nginx 
```
nginx -s start
```
3. 关闭nginx
```bash
nginx -s stop
```
4. 重新加载
```bash
nginx -s reload
```

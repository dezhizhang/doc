### vi和vim编辑器

### 设置行号
```bash
:set nu
```
### 取消设置行号
```bash
:set nonu
```

```bash
G
```
### 一搬模式下gg定位到首行
```bash
gg
```
### 一搬模式下u撤销
```bash
u
```
###  一搬模式下定位到某一行
```bash
n shift+g
```
